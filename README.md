# Miti-LPE
For mitigating Local Privilege Escalation that used `sudo` vulnerability.

# Credit
https://github.com/zinzloun/CVE-2025-32463

# How to run
Clone first
    git clone https://github.com/Raka200juta/Miti-LPE
    cd Miti-LPE

Run script
    :~Miti-LPE$ ./update.sh 
    or
    :~Miti-LPE$ bash update.sh
